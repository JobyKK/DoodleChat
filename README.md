# ReactiveChat

There are two branches:
1) "master" - the implementation with websockets
2) "polling" - the implementation with polling

## FrontEnd
```cd frontend
npm i
npm run dev
```

## BackEnd

```
cd backend
mvn clean install
java -jar
```

## DB
H2 - http://localhost:8080/h2-console/login.jsp
JDBC URL: jdbc:h2:mem:chatdb
